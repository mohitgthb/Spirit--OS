## Features

- Drag and drop windows
- Multi-agent AI assistant
- Multiple input methods (gesture, voice, keyboard)
- Accessibility profiles

## Getting Started

1. Open the File Explorer to browse files
2. Use the AI assistant to control the OS
3. Try gesture or voice commands in Settings
4. Explore the built-in apps: Calculator, Notes, Terminal, Browser

Enjoy SavitaOS!
